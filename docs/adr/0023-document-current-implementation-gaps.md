# Document Current Implementation Gaps

The streaming spec includes a Current Implementation Gaps section that separates shipped Reference Implementation behavior from the target provider-neutral Coding Remote Agent contract. Post-hoc, this section should be treated as a living implementation audit rather than a snapshot of the old knock-knock/status-only demo. As the simulator closes gaps, keep only the remaining concrete divergences there, such as text-only Scenario Matching or missing scenario coverage for a particular lifecycle edge.
