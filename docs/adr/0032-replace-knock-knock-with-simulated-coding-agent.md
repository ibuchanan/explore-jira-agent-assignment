# Replace Knock-Knock With an A2A Simulator

The Reference Implementation should evolve from the thin knock-knock streaming demo into a general A2A Simulator. The simulator is deterministic and does not require a real agent backend, but it emits explicit A2A-compatible task, status, message, and artifact updates from editable scenarios. A Simulated Coding Remote Agent should be one scenario set for the simulator, demonstrating repository inspection, progress and tool activity, optional input or authorization pauses, coding artifacts, and completion.
