Problem

Scenario execution has depth, but the interface is spread across route handlers, cursor writes, active stream state, and task mutation.

Solution

Move playback cursor, first-step folding, pause/final stopping, resubscribe continuation, and cancellation sequencing into one local module.

Wins

  • locality: lifecycle rules concentrate
  • leverage: one scenario seam
  • tests hit one module
  • server becomes adapter
Keeps ADR-0035 intact: the simulator stays inside apps/remote until another consumer proves the seam.
